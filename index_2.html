3things Diary Documentation

System Requirements

  • macOS 12.0 or later
  • 64-bit processor
  • Minimum 50MB free storage

Getting Started

Installation

  1. Download 3things Diary from the Mac App Store
  2. Open the application
  3. Complete the initial setup by entering your name
  4. Start journaling!

Key Features

  • Daily journal entries with rich text formatting
  • "Three Things" focused journaling mode
  • Mood tracking with five emotional states
  • Daily inspirational quotes
  • Automatic entry creation for the past three days

Keyboard Shortcuts

Action Shortcut
New Entry ⌘N
Toggle Sidebar ⌘⇧S
Bold Text ⌘B

Data Management

Backup

Your entries are automatically saved in:

~/Library/Containers/com.matthieudraveny.3thingsdiary/Data/
We recommend regularly backing up this folder to ensure your entries are safe.

Privacy & Security

  • All data is stored locally on your device
  • No data is collected or transmitted
  • Entries are protected by macOS security features
  • Data is stored using Core Data with encryption at rest

Troubleshooting

Common Issues

  • App won't start: Ensure you're running macOS 12.0 or later
  • Missing entries: Check the sidebar filters
  • Text formatting issues: Try selecting all text (⌘A) and clearing formatting
